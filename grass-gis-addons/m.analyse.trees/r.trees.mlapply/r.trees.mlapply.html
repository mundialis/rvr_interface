DESCRIPTION

r.trees.mlapply applies the tree classification model in parallel to the area of interest (current region. The actual classification is performed by r.trees.mlapply.worker

SEE ALSO

r.learn.ml2

EXAMPLES

Application of the tree classification model in parallel

r.trees.mlapply model=ml_trees_randomforest.gz \
                group=ml_input \
                area=area_of_interest \
                output=trees_ml_raw \
                tile_size=100 \
                nprocs=2
