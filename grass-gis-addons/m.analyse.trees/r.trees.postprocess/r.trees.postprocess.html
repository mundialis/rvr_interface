DESCRIPTION

r.trees.postprocess generates single tree delineations from tree pixels and geomorphological peaks. The tree pixels are the result of a machine learning (ML) model to detect trees, see r.trees.mltrain and r.trees.mlapply.

EXAMPLES

Generation of single tree delineations from tree pixels

r.trees.postprocess tree_pixels=mltrees \
                    nearest=trees_nearest \
                    peaks=trees_peaks \
                    red_raster=top.red \
                    green_raster=top.green \
                    blue_raster=top.blue \
                    nir_raster=top.nir \
                    ndvi_raster=top.ndvi \
                    ndwi_raster=top.ndwi \
                    ndgb_raster=top.ndgb \
                    ndsm=ndsm \
                    slope=ndsm_slope \
                    ndvi_threshold=130 \
                    nir_threshold=130 \
                    ndsm_threshold=1 \
                    slopep75_threshold=70 \
                    area_threshold=5
