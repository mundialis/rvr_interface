DESCRIPTION

r.trees.peaks uses object heights from a normalized digital object model (nDOM) to detect peaks, to set uniques IDs for these peaks and to assign each pixel to the nearest peak using a cost analysis with r.cost where inverted slope values are used as costs.

EXAMPLES

Tree peak extraction using default values

r.trees.peaks ndom=ndom_raster nearest=nearest_tree peaks=tree_peaks slope=ndom_slope

SEE ALSO

r.mapcalc, r.cost, r.slope.aspect
