NAME

m.analyse.trees - GRASS GIS addons for detection and comparison of single trees and calculation of various tree parameters for them.

KEYWORDS

raster, vector

DESCRIPTION

The m.analyse.trees toolset consists of several modules.
  • v.tree.param: Calculates various tree parameters for given treecrown vector map
  • v.tree.param.worker: Worker for parallel processing of v.tree.param

REQUIREMENTS

The following Python libraries are needed.
