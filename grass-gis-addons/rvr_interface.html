NAME

rvr_interface - GRASS GIS addons for Regionalverband Ruhr (RVR) related geodata processing.

KEYWORDS

raster, import, vector

DESCRIPTION

The rvr_interface toolset consists of several modules.
  • r.extract.buildings: Extracts buildings from nDOM, NDVI and FNK
  • r.extract.greenroofs: Extracts green roofs from nDOM, NDVI, GB-Ratio, FNK and building outlines
  • r.import.dgm_nrw: Downloads and imports the NRW DGM 1m into the current mapset.
  • r.import.ndom_nrw: Calculates nDOM from DOM and DGM data
  • v.cd.buildings: Calculates difference between two vector layers (buildings)

REQUIREMENTS

The following Python libraries are needed.
  • psutil
  • pyproj
  • requests
  • tqdm
