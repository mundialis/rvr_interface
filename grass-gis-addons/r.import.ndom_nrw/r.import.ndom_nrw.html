DESCRIPTION

r.import.ndom_nrw calculates an nDOM by subtracting input DGM data (defined by the dgm parameter) from an input DOM indicated by the dom parameter. If no DGM is defined, NRW DGM data is automatically imported using r.import.dgm_nrw. Potential NoData areas in the dom are filled beforehand by r.fillnulls and both rasters are resampled to a common grid. The final nDOM is resampled to match the extent and resolution of the current computational region.

EXAMPLE

r.import.ndom_nrw dom=laz_import_dom output_ndom=ndom memory=8000

SEE ALSO

r.import.dgm_nrw, r.fillnulls, r.mapcalc
