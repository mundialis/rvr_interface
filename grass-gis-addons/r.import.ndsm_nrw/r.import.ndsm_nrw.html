DESCRIPTION

r.import.ndsm_nrw calculates an nDSM by subtracting input digital terrain model (DTM) data (defined by the dtm parameter) from an input DSM indicated by the dsm parameter. If no DTM is defined, NRW DTM data is automatically imported using r.import.dtm_nrw. Potential NoData areas in the dsm are filled beforehand by r.fillnulls and both rasters are resampled to a common grid. The final nDSM is resampled to match the extent and resolution of the current computational region.

EXAMPLE

r.import.ndsm_nrw dsm=laz_import_dsm output_ndsm=ndsm memory=8000

SEE ALSO

r.import.dtm_nrw, r.fillnulls, r.mapcalc
