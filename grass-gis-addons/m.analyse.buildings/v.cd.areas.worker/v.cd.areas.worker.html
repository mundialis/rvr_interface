DESCRIPTION

v.cd.areas.worker is a worker module that is started by v.cd.areas. It calculates differences between two vector layers (e.g. classification and reference) by making use of v.overlay with operator "xor".

Optionally, quality measures completeness and correctness can be calculated.

SEE ALSO

v.overlay, v.to.db, v.db.droprow
