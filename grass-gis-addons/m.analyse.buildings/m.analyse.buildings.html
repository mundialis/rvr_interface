NAME

m.analyse.buildings - GRASS GIS addons for detection and comparison of buildings and detection of vegetated roofs on buildings.

KEYWORDS

raster, import, vector

DESCRIPTION

The m.analyse.buildings toolset consists of several modules.
  • r.extract.buildings: Extracts buildings from nDSM, NDVI and FNK
  • r.extract.greenroofs: Extracts green roofs from nDSM, NDVI, GB-Ratio, FNK and building outlines
  • r.import.dtm_nrw: Downloads and imports the NRW digital terrain model (DTM) 1m into the current mapset.
  • r.import.ndsm_nrw: Calculates nDSM from DSM and DTM data
  • v.cd.areas: Calculates difference between two vector layers (buildings)

REQUIREMENTS

The following Python libraries are needed.
  • psutil
  • pyproj
  • requests
  • tqdm
