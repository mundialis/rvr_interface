DESCRIPTION

v.param.tree calculates various tree parameters for tree crowns given as input vector map einzelbaeume. The module returns following parameters: tree height, crown area, crown perimeter, NDVI per single tree, crown volume, stem position, distance to nearest building and distance to nearest tree. The module takes as input a raster map of the ndom for computation of the tree height, a raster map of the ndvi for computation of the NDVI per single tree and a vector map of the gebaeude for computation of the distance to the nearest building. Optional the parameter abstand_baum_umkreis can be given, determining the range within neighbouring trees are searched for.

EXAMPLE

Example 1

v.param.tree ndom=ndom ndvi=ndvi gebaeude=hausumringe einzelbaeume=trees_ml

Example 2: with optional input of abstand_baum_umkreis

v.param.tree ndom=ndom ndvi=ndvi gebaeude=hausumringe einzelbaeume=trees_ml abstand_baum_umkreis=500
