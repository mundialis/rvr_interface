DESCRIPTION

v.tree.param calculates various tree parameters for tree crowns given as input vector map einzelbaeume. The module returns following parameters: tree height, crown area, crown perimeter, NDVI per single tree, crown volume, stem position, distance to nearest building and distance to nearest tree. The module takes as input a raster map of the ndom for computation of the tree height, a raster map of the ndvi for computation of the NDVI per single tree and a vector map of the gebaeude for computation of the distance to the nearest building. Optional the parameters abstand_gebaeude_umkreis and abstand_baum_umkreis can be given, determining the range within neighbouring buildings or trees are searched for. First the region is set to the tree of interest, then the region ist extended in each direction by abstand_gebaeude_umkreis or abstand_baum_umkreis (given in meters). By default abstand_gebaeude_umkreis is unset, and abstand_baum_umkreis is set to 500 m.

EXAMPLE

Example 1

v.tree.param ndom=ndom ndvi=ndvi gebaeude=hausumringe einzelbaeume=trees_ml

Example 2: with optional input of abstand_baum_umkreis

v.tree.param ndom=ndom ndvi=ndvi gebaeude=hausumringe einzelbaeume=trees_ml abstand_baum_umkreis=500
