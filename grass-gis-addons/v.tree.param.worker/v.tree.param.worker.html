DESCRIPTION

v.tree.param calculates various tree parameters for tree crowns given as input vector map treecrowns. The module returns following parameters: tree height, crown area, crown perimeter, NDVI per single tree, crown volume, stem position, distance to nearest building and distance to nearest tree. The module takes as input a raster map of the ndom for computation of the tree height, a raster map of the ndvi for computation of the NDVI per single tree and a vector map of the buildings for computation of the distance to the nearest building. Optional the parameters distance_building and distance_tree can be given, determining the range within neighbouring buildings or trees are searched for. First the region is set to the tree of interest, then the region ist extended in each direction by distance_building or distance_tree (given in meters). By default distance_building is unset, and distance_tree is set to 500 m.

EXAMPLE

Example 1

v.tree.param ndom=ndom ndvi=ndvi buildings=hausumringe treecrowns=trees_ml

Example 2: with optional input of distance_tree

v.tree.param ndom=ndom ndvi=ndvi buildings=hausumringe treecrowns=trees_ml distance_tree=500
